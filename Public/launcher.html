Before you can use the widget, you must authorize it. Click the "Authorize" button to open the Auth window. Once you've finished, the Auth window will close and you can then click Launch. The demo will be around whether or not you authorize.

Click "Launch Widget" to launch the live widget in your current browser. If Bit Boss Battles won't work properly in your streaming software as a web source, this is an alternate solution. If you'd like to try Bit Boss Battles first, you can click "Launch Demo" to see how it works. NOTE: The demo ignores all settings except sound.

Click "Set Bit Boss" to clear the current bit boss and set the new bit boss with the listed values.

Click "Reset Cookies" to clear the widget's cookies. This clears the current boss, their health, and resets all options. NOTE: This only clears the cookies in the current browser. To clear cookies when using the URL for stream software, temporarily disable Persistent Mode in Settings. If you never run Persistent Mode, then cookies are always cleared on startup.
