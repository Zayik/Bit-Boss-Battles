Bit Boss Battles (BBB) is an open source, viewer interactive bits widget for the Twitch platform. Streamers using BBB enable their viewers to fight each other using bits to become the next Bit Boss.

This adaptation of Bit Boss Battle adds additional features as well as incorporating channel points.

If you'd like to peek at this version of the BBB source code, or report an issue, visit this BBB's GitHub page.

Or, if you'd like to support this open source adaptation of BBB, head on over to BiffMasterZay's Twitch page.
